Kullanım

Aşağıda sınıfın örnek bir kullanımını görebilirsiniz. Genelde 4 basit adımda sonuçlara ulaşabilirsiniz. Aşağıdaki örnekte "uyeler" isimli tablodan yaşı 50'den küçük olan bütün üyeleri alıyoruz ve sonuçlara gözatmış oluyoruz.

<?php
    // 1. dosyaları ve seçenekleri yükle
    require_once 'dbo.class.php';
    $opt = require 'dbo.opt.php';

    // 2. sınıfı kullanıma hazırla
    $dbo = DBO::getInstance();
    $dbo->setOpt($opt);
    $dbo->connect_db();

    // 3. veritabanına sorgu gönder
    $dbo->setSql('SELECT * FROM `uyeler` WHERE `yas` < 50');
    $dbo->runSql();

    // 4. sonuçları ekranda göster
    $dbo->dump( $dbo->getAll() );
?>

İPUCU: dosyaların yüklenmesi, seçeneklerin belirlenmesi ve sınıfın kullanıma hazırlanması ile ilgili satırları her sayfaya yazmakla uğraşmamak için, bu satırları bir dosyaya 1 kereliğine yazabilirsiniz (örneğin dbo.init.php gibi):

<?php
    // db.init.php dosyasının içeriği
    require_once 'dbo.class.php';
    $opt = require 'dbo.opt.php';
    $dbo = DBO::getInstance();
    $dbo->setOpt($opt);
    $dbo->connect_db();
    unset($opt);
?>

Daha sonra bu dosyayı ilgili sayfa içinden çağırarak sınıfı kullanmaya hemen başlayabilirsiniz. Örnek: 

<?php
    // hızlı başlangıç dosyasını yükle
    require_once 'dbo.init.php';

    // veritabanına sorgu gönder
    $dbo->setSql('SELECT * FROM `uyeler` WHERE `yas` < 50');
    $dbo->runSql();

    // sonuçları ekranda göster
    $dbo->dump( $dbo->getAll() );
?>
